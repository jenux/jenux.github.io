Write your blog with Octopress and Git

Tools Blog 2013/11/01

本文于 2666 天之前发表，文中内容可能已经过时。

Octopress is a blogging framework designed for Jekyll, the blog aware static site generator powering Github Pages.
As you know, you can set up a free blog base on Github Pages service, and using octopress, you can easily set up it.
Just simply clone or fork Octopress, install dependenciesand the theme, and you’re set.

You should be comfortable running shell commands and familiar with the basics of Git.

Required

  • Install Git
  • Install Ruby 1.9.3 or greater using either rbenv or RVM.
    If ruby –version doesn’t say you’re using Ruby at least 1.9.3, revisit your rbenv or RVM installation.

Get Started

Octopress is easy to set up and deploy.

  1. clone git repository
  2. Install dependencies
  3. Install default Octopress theme

    Set up deployment for Github

    Hosting your blog with Github’s Pages service is free and allows custom domains.
    To deploy you simply push your repository to Github.

Deploy also support Heroku / Rsync

  1. Create a new Github repository and name the repository with the format username.github.io,
    where username is your GitHub user name or organization name.

  2. config

    Configure your blog

    The rake task will ask you for a URL of the Github repo.
    Copy the SSH or HTTPS URL from your newly created repository (e.g. git@github.com:username/username.github.io.git) and paste it in as a response.

Don’t forget to commit the source for your blog.
