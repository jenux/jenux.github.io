Introducing HTML5 FullScreen API

Notes HTML5 2013/07/24

本文于 2766 天之前发表，文中内容可能已经过时。

Fullscreen API

Html5提供了Fullscreen API, 帮助开发者控制浏览器的全屏模式。

Note:
最新版文档描述的属性和方法大小写与之前有所不同，经测试Chrome (27.0.1453.116)均支持不同的大小写方式, Firefox (22)不支持最新文档写法。

element.requestFullscreen()

以全屏模式显示批定Dom元素。 [注: 前版为 element.requestFullScreen()]

document.exitFullscreen()

退出全屏模式。 [注: 前版为 element.cancelFullScreen()]

Fullscreen CSS

其他属性

  • document . fullscreenEnabled
    true/false, document是否支持显示全屏
  • document . fullscreenElement
    返回全屏DOM元素
