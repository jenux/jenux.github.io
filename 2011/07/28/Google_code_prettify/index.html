为任意项目甚至html静态页面增加代码高亮的插件

Tools Javascript Plugin 2011/07/28

本文于 3494 天之前发表，文中内容可能已经过时。

这个教程运用 Google code prettify 简单快捷地为任意项目，甚至是单纯的HTML静态页面添加代码高亮。

#第1步: 下载源码
下载源码: Google code

#第2步: 把src目录拷进你的项目
我习惯把src目录重命名为: highlighter

#第3步: 在项目页面中引用文件
在你的页面中引用prettify.css和prettify.js

<!Doctype html>
<html>
    <head>
        <meta charset='utf-8'>
        <title>Google code prettify demo</title>
        <link rel="stylesheet" href="highlighter/prettify.css">
        <script src='highlighter/prettify.js'></script>
    </head>
    <body>
        
    </body>
</html>

代码高亮插件将对带有’prettyprint’样式的 pre|code标签起作用。
