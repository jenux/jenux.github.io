Simiki-简单的个人Wiki框架

Tools Wiki 2018/01/22

本文于 1123 天之前发表，文中内容可能已经过时。

Simiki 是一个简单的个人Wiki框架。使用Markdown书写Wiki, 生成静态HTML页面。Wiki源文件按目录分类存放, 方便管理维护。

Install

pip install simiki

Generate
