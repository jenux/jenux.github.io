推荐2个Markdown编辑器

Tools Markdown 2014/06/24

本文于 2431 天之前发表，文中内容可能已经过时。

小众软件有一个专题收集Markdown系列应用, 先推荐两个不错的应用

Markable.in
一个在线编辑器，可以导入Dropbox, Evernote的文章进行编辑，编辑结束后再导出保存到原文件。

马克飞象
马克飞象是一款专为印象笔记（Evernote）打造的Markdown编辑器, 功能比Markable强大很多，如:

  • 丰富的编辑器主题设定
  • 键盘模式可选择Vim 或 Emacs
  • 有离线客户端 Chrome APP
  • 马克印象保存的文章右上角有标记，点击直接调用离线客户端进行编辑
  • 支持LaTex公式

专业版目前处于公测阶段，未来可能进行收费。
